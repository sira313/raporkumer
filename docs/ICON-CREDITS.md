# Kredit & Sumber Ikon

Ikon yang ada di `src/lib/icons` diambil dari proyek Feather (Feather Icons):

Repository: https://github.com/feathericons/feather

Feather Icons dirilis di bawah lisensi MIT. Penggunaan ikon di luar konteks internal (mis. pengemasan ulang,
distribusi tersendiri) harus mematuhi syarat lisensi MIT.

Rekomendasi:
- Sertakan atribusi sederhana saat menggunakan ikon di tempat publik: "Ikon oleh Feather (https://feathericons.com), lisensi MIT".
- Untuk audit lisensi, periksa file `LICENSE` pada repo Feather di tautan di atas.
