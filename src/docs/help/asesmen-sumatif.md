---
title: Petunjuk
---

Cukup klik tombol <kbd class="kbd">Nilai</kbd> pada kolom aksi untuk melakukan penilaian pada masing-masing murid.<br>
Bila ingin mengisi nilai melalui Excel, gunakan tombol <kbd class="kbd">Download Template</kbd> untuk mengunduh template excel. Jangan ubah template tersebut, cukup isi nilai saja. Pastikan tidak ada rumus.<br>
Setelah selesai mengisi nilai, import kembali melalui tombol <kbd class="kbd">Import Nilai</kbd>.
