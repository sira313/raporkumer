Halaman ini berfungsi untuk menambahkan mata evaluasi keasramaan bagi sekolah yang menyelenggarakannya.

Tombol <kbd class="kbd">Kelola Mata Evaluasi</kbd> berguna untuk menambahkan, mengedit, dan menghapus mata evaluasi beserta indikatornya.

Tombol <kbd class="kbd">Edit TP</kbd> digunakan untuk menambahkan, edit, dan hapus tujuan pembelajaran masing-masing indikator.
